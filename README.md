# POS Pantau PG Rendeng

Sistem monitoring penerimaan tebu di 6 pos pantau PG Rendeng Kudus.

Stack: Laravel 11 + Sanctum + PostgreSQL/MySQL, Vanilla HTML + Tailwind PWA, Leaflet, Chart.js, Fonnte WhatsApp.
